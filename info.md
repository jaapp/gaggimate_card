# GaggiMate Card

A beautiful and functional custom Lovelace card for controlling your GaggiMate espresso machine directly from Home Assistant.

## Features

✅ Real-time temperature and pressure monitoring
✅ Quick mode switching (Standby, Brew, Steam, Water)
✅ Brew start/stop control
✅ Active profile display
✅ Progress tracking with visual feedback
✅ Full theme support (light/dark)

## Prerequisites

This card requires the [GaggiMate Home Assistant integration](https://github.com/jaapp/ha_gaggimate) to be installed and configured.
